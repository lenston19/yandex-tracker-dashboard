::accordion

defaultValue:

::accordion-item{label="Ошибка 401 — что это значит?" icon="i-lucide-circle-help"}
Ошибка **401 (Unauthorized)** возникает, когда система не может подтвердить ваши права доступа.  
Чаще всего это происходит, если вы вошли в систему с аккаунта, который **не связан с вашей организацией**.
::

::accordion-item{label="Как исправить ошибку 401?" icon="i-lucide-circle-help"}
Иногда пользователь случайно авторизуется под другим аккаунтом, из-за чего система отклоняет запросы.  
Чтобы решить проблему, выполните следующие шаги:

1. Перейдите на страницу  
   👉 [**Доступ к данным аккаунта**](https://id.yandex.ru/personal/data-access){target=_blank}  
2. Найдите приложение, которому был выдан доступ.  
3. Отзовите (удалите) его разрешения.  
4. Вернитесь на сайт и повторите авторизацию под нужным аккаунтом.
::

::accordion-item{label="Можно ли просто очистить cookies?" icon="i-lucide-circle-help"}
Иногда помогает — очистите cookies и перезапустите браузер.
> Если удалить куки, то все настройки сбросятся
::

## 🗂 Мои задачи

::accordion

::accordion-item{label="Какие задачи попадают в список?" icon="i-lucide-circle-help"}
В список попадают задачи, в которых вы указаны в одной из выбранных ролей:

- **Исполнитель** — задача назначена на вас
- **Ревьювер** — вы указаны в поле «Ревьювер»
- **QA** — вы указаны в поле «QA-инженер»

Роли настраиваются в разделе [**Настройки**](/settings){target=_blank} → «Мои задачи».
::

::accordion-item{label="Почему не отображаются задачи из нужной очереди?" icon="i-lucide-circle-help"}
Проверьте фильтр **«Очередь»** на странице — возможно, указана конкретная очередь, и задачи из других очередей скрыты.  
Очистите поле очереди, чтобы увидеть задачи из всех очередей.

Также убедитесь, что нужные **статусы** включены в фильтр.
::

::accordion-item{label="Что означают цвета статусов?" icon="i-lucide-circle-help"}
Цвет бейджа статуса отражает тип состояния задачи:

- 🔵 **Синий** — в работе
- 🟦 **Голубой** — открыта
- 🟢 **Зелёный** — закрыта / решена
- 🔴 **Красный** — отклонена / отменена
- 🟡 **Жёлтый** — нужна информация / переоткрыта
- 🟣 **Фиолетовый** — на тестировании / ревью
::

## ⏱ Таймер

::accordion

::accordion-item{label="Как запустить таймер?" icon="i-lucide-circle-help"}
Откройте карточку задачи и нажмите кнопку **«Запустить таймер»**.  
Таймер отсчитывает время и сохраняет ворклог автоматически при остановке.

Также можно найти задачу через поиск прямо в виджете таймера на дашборде.
::

::accordion-item{label="Можно ли запустить несколько таймеров одновременно?" icon="i-lucide-circle-help"}
Нет — одновременно может работать только **один таймер**.  
При запуске нового таймера предыдущий необходимо сначала остановить.
::

::accordion-item{label="Ворклог не появился после остановки таймера — что делать?" icon="i-lucide-circle-help"}
1. Проверьте, что период на дашборде включает сегодняшний день.
2. Убедитесь, что в момент остановки было активное интернет-соединение.
3. Обновите страницу — ворклог мог не отобразиться из-за кэша.

Если проблема повторяется, попробуйте добавить ворклог вручную через интерфейс Yandex Tracker.
::

::

## 📊 Ворклоги и виджеты

::accordion

::accordion-item{label="Почему виджет пустой / данные не загружаются?" icon="i-lucide-circle-help"}
Возможные причины:

- Истёк OAuth-токен — попробуйте выйти и войти снова.
- Выбранный период не содержит ворклогов — смените диапазон дат.
- Нет доступа к организации — уточните Organization ID в [**Настройках**](/settings){target=_blank}.
::

::accordion-item{label="Можно ли редактировать ворклог?" icon="i-lucide-circle-help"}
Редактирование ворклогов доступно напрямую через интерфейс [**Yandex Tracker**](https://tracker.yandex.ru){target=_blank}.  
В этом приложении ворклоги возможно только **удалять** и **обновлять время и статус** через таймер.
::

::accordion-item{label="Как изменить отображаемый период?" icon="i-lucide-circle-help"}
На дашборде используйте переключатель периода в верхней части страницы.  
Доступны режимы: **день**, **неделя**, **месяц**.  

Для месячного отчёта перейдите на страницу [**Отчёт**](/monthly-report){target=_blank}.
::

::

## ⚙️ Общее

::accordion

::accordion-item{label="Где взять Organization ID?" icon="i-lucide-circle-help"}
Organization ID можно найти на странице вашей организации в Yandex Cloud:

👉 [**center.yandex.cloud**](https://center.yandex.cloud){target=_blank}

Выберите нужную организацию — ID отображается в адресной строке и в настройках организации.
::

::accordion-item{label="Поддерживается ли мобильный браузер?" icon="i-lucide-circle-help"}
Да — интерфейс адаптирован для мобильных устройств.  
Рекомендуется использовать актуальную версию Chrome или Safari.
::

::accordion-item{label="Данные хранятся на серверах сервиса?" icon="i-lucide-circle-help"}
Нет. Приложение работает напрямую между вашим браузером и Yandex Tracker API.  
Никакие данные (ворклоги, токены, настройки) не передаются на сторонние серверы.  
Настройки сохраняются локально в cookies вашего браузера.
::

::

# Что нового?

## [1.1.10] - 2026-03-16

### Fixed
- `RangeError: Invalid time value` при форматировании дат у пользователей со старым форматом cookie временной зоны — добавлена автоматическая миграция при старте приложения

### Added
- CI/CD: автоматический релиз при merge в `master`

## [1.1.9] - 2026-03-15

### Added
- Виджет «График активности» — Heatmap с выбором периода (3, 6, 9 месяцев или 1 год)
- Российский производственный календарь через `isdayoff.ru` — учёт праздников и переносов
- Недельный виджет теперь отмечает выходные и праздничные дни, пересчитывает норму часов
- Сводка месяца учитывает рабочие дни по производственному календарю РФ

### Changed
- Централизованное форматирование дат через новый composable `useDateFormatter` с поддержкой выбранной пользователем временной зоны
- Миграция с `luxon` на `date-fns` + `date-fns-tz` для работы с датами и временными зонами
- Пагинация вынесена в универсальную утилиту `fetchAllPages` — убрано дублирование кода
- Страница настроек полностью переработана

### Fixed
- Исправлена утечка памяти: URL аватара пользователя теперь освобождается автоматически при размонтировании компонента
- Исправлен баг с состоянием загрузки: `startLoading`/`finishLoading` вызывались при инициализации вместо того, чтобы быть функциями-обработчиками
- Ошибка `value > max` в прогресс-баре при переработках сверх нормы
